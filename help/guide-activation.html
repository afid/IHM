⚡ Guide Utilisateur : Activer une Situation

Ce module vous permet de lier des situations créées précédemment aux différentes structures de votre arborescence (Marques, Domaines ou Sous-domaines).

1. Naviguer dans l'arborescence

Pour activer une situation sur une structure spécifique, vous devez naviguer du haut vers le bas.

  • Sélectionnez d'abord une Marque.
  • Puis affinez en sélectionnant un Domaine (optionnel).
  • Et enfin un Sous-domaine (optionnel).
Héritage : Gardez à l'esprit que si vous activez une situation sur une Marque, celle-ci s'appliquera par défaut à tous les Domaines et Sous-domaines rattachés.

2. Gérer les situations d'une structure

Une fois votre structure sélectionnée, si elle existe déjà dans le système, ses situations actives s'afficheront. Cliquez sur Modifier l'Activation pour ajouter ou retirer des situations pour cette structure.

3. Ajouter ou retirer une situation

Dans la fenêtre d'édition :

  • Ajouter : Sélectionnez une situation dans la liste déroulante puis cliquez sur "Ajouter la situation". Un récapitulatif des périodes s'affichera pour confirmation.
  • Retirer : Cliquez sur la croix (❌) à côté d'une situation active pour la retirer de la structure.
  • Sauvegarder : N'oubliez pas de cliquer sur "Sauvegarder les modifications" pour appliquer vos changements.

4. Créer une nouvelle combinaison de structure

Si une structure n'est pas encore déclarée, utilisez le bouton ➕ Nouvelle Structure pour enregistrer la combinaison Marque/Domaine/Sous-domaine et procéder à l'affectation de ses situations.
