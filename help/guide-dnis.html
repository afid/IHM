Guide Utilisateur : Paramétrage des DNIS 📞

Tout ce que vous devez savoir pour gérer vos numéros d'appel.

🚨 Pré-requis technique

L'usage d'un serveur HTTP est obligatoire pour faire fonctionner cette interface. Ne tentez pas d'ouvrir ce fichier directement via votre explorateur de fichiers (double-clic).

1. C'est quoi un DNIS ?

Imaginez le DNIS (Dialed Number Identification Service) comme l'adresse que le client tape dans son GPS pour arriver chez vous. Concrètement, c'est le numéro de téléphone que le client a composé.

Ce module vous permet de définir ce qui se passe quand quelqu'un appelle ce numéro. C'est ici que vous décidez : "Si quelqu'un appelle le 01 23 45 67 89, on considère que c'est pour la Marque A, et on lui joue tel message."

2. L'Interface en un coup d'œil

L'écran est divisé en deux parties principales :

  • La zone de sélection (en haut) : C'est ici que vous choisissez sur quel numéro vous voulez travailler. Vous avez une liste déroulante qui affiche tous vos numéros existants.
  • La zone de boutons :
    • Create Nouveau DNIS : Pour créer un tout nouveau numéro.
    • Edit Editer : Pour modifier le numéro sélectionné.
    • Duplicate Dupliquer : Pour copier le numéro sélectionné (pratique pour gagner du temps !).
    • Delete Supprimer : Pour effacer définitivement un numéro (Attention !).

3. Comment faire...

Créer un nouveau numéro

  1. Cliquez sur le bouton ➕ Nouveau DNIS.
  2. Une fenêtre s'ouvre. Remplissez au minimum le Numéro (doit commencer par +33...) et la Marque.
  3. Remplissez les autres informations si vous les avez (voir section suivante).
  4. Cliquez sur Sauvegarder.

Modifier un numéro existant

  1. Sélectionnez le numéro dans la liste déroulante.
  2. Vérifiez les détails qui s'affichent pour être sûr que c'est le bon.
  3. Cliquez sur ✏️ Editer.
  4. Changez ce qui doit l'être. Note : Vous ne pouvez pas changer le numéro lui-même une fois créé.
  5. Cliquez sur Sauvegarder.

Dupliquer un numéro (Copier-Coller intelligent)

Vous avez un nouveau numéro qui fonctionne presque pareil qu'un ancien ? Ne recommencez pas de zéro !

  1. Sélectionnez le numéro "modèle" dans la liste.
  2. Cliquez sur 📄 Dupliquer.
  3. La fenêtre de création s'ouvre avec tout pré-rempli.
  4. Important : Saisissez le Nouveau Numéro (le champ est vide par sécurité).
  5. Modifiez ce qui change, puis cliquez sur Sauvegarder.

4. Comprendre les champs

Voici à quoi sert chaque case du formulaire :

Numéro (DNIS) C'est l'identifiant unique. Il doit être au format international, par exemple +33123456789.
Marque Pour quelle entité le client appelle-t-il ? (Ex: Covea, MAAF, GMF...). C'est essentiel pour le reporting. L'interface propose désormais un menu déroulant pour garantir l'utilisation de noms standardisés.
Domaine / Sous-Domaine / Nature / etc. Regroupements métiers permettant de catégoriser finement l'appel. Ces champs utilisent également des menus déroulants pour faciliter la saisie et éviter les erreurs.
Nom Un petit nom pour vous y retrouver plus facilement dans la liste (Ex: "Ligne VIP", "Accueil Sinistre").
Voix Acteur Qui va parler au client ? Sélectionnez la voix de synthèse ou l'acteur (Mathieu, Lea, Celine, Remi).
Ani Simulation / Date Simulation Champs techniques utilisés pour tester vos parcours. Laissez vide si vous ne savez pas.
Logger Actif Cochez cette case si vous voulez enregistrer des traces détaillées pour ce numéro (utile pour le débogage).

5. Les Modules (Le "Cerveau")

Les modules définissent les actions spécifiques à exécuter lors de l'appel. Par exemple : "Jouer une annonce", "Vérifier un calendrier", "Router l'appel".

Ajouter un module

  1. Dans la section Modules en bas du formulaire.
  2. Choisissez un type de module dans la liste (ex: Module_Play_Announcement).
  3. Cliquez sur Ajouter.

Configurer un module

Une fois ajouté, le module apparaît sous forme de "carte".

  • Choisir une action : Sélectionnez ce que le module doit faire dans le menu déroulant.
  • Actions multiples : Certains modules (comme les vérifications) permettent de faire plusieurs choses à la suite.
    • Cliquez sur le petit bouton vert ➕ Ajouter une action pour en ajouter une autre.
  • Supprimer : Si vous vous êtes trompé, cliquez sur la petite poubelle 🗑️ pour retirer le module ou la croix ✖️ pour retirer une ligne d'action.

🚨 L'ordre des modules est important ! Vous pouvez les glisser-déposer pour changer leur ordre d'exécution.
