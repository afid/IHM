🎯 Guide Utilisateur : Gestion des Segments

Ce module permet de configurer les segments de distribution et les modules de ciblage associés.

1. Définition du segment

Un segment est défini par son nom, son type et son groupement.

  • Nom : L'identifiant du segment.
  • Groupement : Obligatoire pour hiérarchiser les segments et permettre l'héritage des propriétés.
  • Type : Catégorie du segment pour le moteur de distribution.

2. Modules de Ciblage

Vous pouvez associer des modules spécifiques pour affiner le ciblage des appels.

  • Modules Pre Ciblage : Exécutés avant le choix final du segment.
  • Modules Post Ciblage : Exécutés après la sélection du segment, pour des actions de finition ou de routage spécifique.
Gestion : Vous pouvez ajouter, modifier ou supprimer des modules dans chaque section. L'ordre des modules peut être réorganisé si nécessaire.

3. Calendriers associés

Un segment peut être lié à un ou plusieurs calendriers pour définir ses périodes d'activité. Utilisez le bouton "Ajouter un calendrier" pour lier un calendrier existant au segment.
